# Sudoku

A basic Sudoku package providing basic features, such as the ability to solve, generate, and manipulate Sudoku puzzles.

## Getting Started

Install Sudoku using [`npm`](https://www.npmjs.com/package/@d-livingston/sudoku):

```bash
npm install @d-livingston/sudoku
```

Including it using Commonjs:

```javascript
const Sudoku = require("@d-livingston/sudoku");
```

Including it using ES6:

```javascript
import Sudoku from "@d-livingston/sudoku";
```
